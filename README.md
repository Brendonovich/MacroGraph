An application for node-based programming.

## Running

1. Make sure you have [Tauri](https://tauri.app/v1/guides/getting-started/prerequisites) and [pnpm](https://pnpm.io/) installed.
2. Run `pnpm i`
4. Run `pnpm app tauri dev`
